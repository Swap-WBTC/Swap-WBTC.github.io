What is Swap WBTC?

Uniswap is a self-custody app destination for trading ERC-20 assets directly from a connected wallet. WBTC is Wrapped Bitcoin on Ethereum, and its token contract can be checked on Etherscan.

How Swap WBTC works

  1. Open the routeUse the app link to open Uniswap with WBTC already selected as the output token.
  2. Connect on EthereumConnect a compatible self-custody wallet and confirm you are using Ethereum mainnet.
  3. Choose the inputSelect ETH or another ERC-20 token, enter the amount, and verify the WBTC contract address.
  4. Review and signCheck the quote, route, price impact, gas, and any token approval before confirming in your wallet.

Before you swap

Token verification

WBTC uses the Ethereum token contract 0x2260FAC5E5542a773Aa44fBCfeDf7C193bc2C599; verify the address before approving or signing.

Self-custody flow

This landing page never takes deposits, custody, seed phrases, or private keys; the transaction is reviewed and signed from your wallet in the app.

Swap WBTC FAQ

Where can I swap wbtc?

You can use the direct Uniswap app route on this page, then confirm the token, route, and final wallet transaction inside the official interface.

Is swap wbtc the same as wrapping BTC?

No. A market trade exchanges one token for WBTC, while wrapping or redeeming BTC involves the WBTC mint and burn process described in the WBTC whitepaper.

What network should I use to swap wbtc?

This route is for WBTC on Ethereum mainnet. Always check the network and token address before approving a transaction.

Why verify the contract before I swap wbtc?

Token symbols can be copied by unrelated contracts, so the Ethereum WBTC address should match 0x2260FAC5E5542a773Aa44fBCfeDf7C193bc2C599.

Does this page charge fees to swap wbtc?

No. This page is only a routing page; network gas, pool fees, price impact, and any app-displayed costs are shown in the swap interface before signing.
